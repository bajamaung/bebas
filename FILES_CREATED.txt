                      DATABASE STRUCTURE SUMMARY

7 Tables Total:

users (7 columns, 3 default rows)
  - Stores user accounts with roles & status
  - Bcrypt hashed passwords
  - 3 demo accounts: admin, petugas, peminjam

kategori (4 columns)
  - Equipment category classifications
  - Name & description
  - Relationships to alat table

alat (10 columns)
  - Equipment inventory
  - Auto-generated codes (ALT + YYYYMMDD + seq)
  - Stock, condition, location tracking
  - Foreign key to kategori

peminjaman (8 columns)
  - Loan requests & orders
  - Auto-generated loan codes (PJM + YYYYMMDD + seq)
  - Status tracking (pending, disetujui, sedang_dipinjam, sudah_dikembalikan, ditolak)
  - Foreign key to users

detail_peminjaman (4 columns)
  - Many-to-many relationship
  - Loan items & quantities
  - Foreign keys to peminjaman & alat

pengembalian (7 columns)
  - Return records
  - Equipment condition on return
  - Fine amounts
  - Return notes
  - Foreign key to peminjaman

log_aktivitas (9 columns)
  - Complete audit trail
  - User, activity type, table, record_id
  - IP address tracking
  - Timestamps
  - Foreign key to users
